Landing Page Template 12-Column Grid System

Für visuell ansprechende Marketing- und Kampagnenseiten

Template-Struktur

HEADER Navigation (12/12 columns)
HERO Kampagne Visual + 2 CTAs (12/12 columns)
VORTEILE 3 Karten nebeneinander (3x 4/12 columns)
PRODUKTE 5 Tarif-Karten (scrollbar bei Overflow)
PROMO Sonderangebot (12/12 columns)
FOOTER 4-spaltig (4x 3/12 columns)

📐 Grid System Erklärung

Alle Templates nutzen das gleiche 12-Column Grid!

• 12/12 = Volle Breite
• 8/12 = 2/3 Breite (gut für Text)
• 4/12 = 1/3 Breite (für Cards)
• 3/12 = 1/4 Breite (für Footer)

Anforderungen

  • Genau 1x H1 pro Seite
  • Max. 2 CTAs im Hero
  • Responsive Design pflicht
  • Alt-Texte für alle Bilder

Best Practices

  • Hero-Headline max. 60 Zeichen
  • Klare Call-to-Actions
  • Trust-Elemente einbauen
  • Mobile-First Design

Service/Prozess Template 12-Column Grid System

Für Formulare und schrittweise Prozesse - Content nutzt 8/12 columns für optimale Lesbarkeit

Template-Struktur

HEADER Navigation (12/12)
TITEL H1 + Einleitung (8/12 centered - 2/12 margin links, 2/12 margin rechts)
FORMULAR Eingabefelder (6/12 centered)
ANLEITUNG 3 Schritte (8/12 centered)
FAQ Akkordeon (8/12 centered)
FOOTER Kompakt (12/12)

📐 Wichtig: Gleiche Grid-Basis!

Dieses Template nutzt dasselbe 12-Column Grid. Der Content wird nur schmaler dargestellt (8/12) für bessere Lesbarkeit bei Text-lastigen Inhalten.

Anforderungen

  • Progressive Disclosure
  • Pflichtfelder markieren
  • Fehlerbehandlung klar
  • ARIA Labels für Forms

Best Practices

  • Max. 5 Felder pro Schritt
  • Fortschrittsanzeige
  • Inline-Validierung
  • Hilfetexte bei Bedarf

Produktübersicht Template 12-Column Grid System

Für Tarifübersichten und Kategorieseiten mit Produktkarten

Template-Struktur

HEADER Navigation (12/12)
INTRO H1 + Beschreibung (12/12)
FILTER Tab-Navigation (12/12)
PRODUKTE 5-Spalten Grid mit Produktkarten
TRUST Awards & Zertifikate (12/12)
FOOTER Extended Footer (12/12)

Anforderungen

  • Filter-Funktionalität
  • Einheitliche Card-Höhe
  • Preise prominent
  • Vergleichbarkeit

Best Practices

  • Max. 5 Produkte nebeneinander
  • Horizontaler Scroll bei mehr
  • Wichtigste Features oben
  • CTA pro Karte

Content/Artikel Template 12-Column Grid System

Für Ratgeber, Hilfeseiten und rechtliche Inhalte - Text in 8/12 für Lesbarkeit

Template-Struktur

HEADER Navigation (12/12)
HERO BILD Optional - 16:9 Ratio (12/12)
ARTIKEL H1 + Content (8/12 centered)
CALLOUT Tip/Info Box (8/12 centered)
RELATED Verwandte Produkte (12/12)
FOOTER Compact Footer (12/12)

Anforderungen

  • H1 → H2 → H3 Hierarchie
  • Max. 700px Textbreite
  • Metadaten anzeigen
  • Lesezeit angeben

Best Practices

  • Kurze Absätze
  • Zwischenüberschriften
  • Listen verwenden
  • Verwandte Links

Produktdetail Template 12-Column Grid System

Für einzelne Tarife und Produktseiten mit Conversion-Fokus

Template-Struktur

HEADER Navigation (12/12)
PRODUKT HERO Name + Key Features (12/12)
PREIS Preis + CTA (12/12)
FEATURES Checkmark Liste (8/12 centered)
TABS Details/Bedingungen (12/12)
ADD-ONS Zusatzoptionen (12/12)
FOOTER Extended Footer (12/12)

Anforderungen

  • Preis prominent
  • Mehrere CTAs
  • Alle Features listen
  • Cross-Selling

Best Practices

  • Above-the-fold CTA
  • Social Proof
  • Klarheit über Kosten
  • Download-Links

Pre-AEM Checkliste

Prüfen Sie diese Punkte bevor Sie in AEM arbeiten

Content Vorbereitung

  • Alle Texte fertig und freigegeben
  • Bilder in richtigen Formaten (WebP/JPEG)
  • Alt-Texte für alle Bilder vorbereitet
  • Meta-Beschreibungen geschrieben

Template-Konformität

  • Richtiges Template gewählt
  • Nur 1x H1 pro Seite
  • Hierarchie H1→H2→H3 eingehalten
  • 12-Column Grid verstanden

Grid-System Check

  • Alle Sections nutzen 12-Column Grid
  • Text-Content in 8/12 für Lesbarkeit
  • Cards in 3/12 oder 4/12
  • Keine 10-Column Layouts verwendet

Accessibility

  • ARIA Labels definiert
  • Kontraste geprüft (WCAG 2.1 AA)
  • Keyboard-Navigation möglich
  • Screenreader-kompatibel
